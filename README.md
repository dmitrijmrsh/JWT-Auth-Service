# jwt-auth-service

Сервис аутентификации и авторизации по JWT. Авторизация реализована в виде ролевой модели 
(админ, менеджер, пользователь). Также добавлены привилегии пользователя для определения доступа при интеграции с
другими сервисами.

Стек технологий:
- Spring Boot
- Spring Security
- Spring Data
- PostgreSQL
- Liquibase
- JUnit5
